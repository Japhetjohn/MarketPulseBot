# MarketPulse Bot
A Telegram bot for real-time crypto analytics using Vybe APIs.

## Features
- /pulse <token>: Live token data.
- /alert <token> <condition>: Custom alerts (e.g., volume>50).
- /history <token>: Historical trends.

## Setup
1. Clone: git clone https://github.com/japhetjohn/MarketPulseBot
2. Install: pip install -r requirements.txt
3. Add .env with BOT_TOKEN and VYBE_API_KEY.
4. Run: python bot.py

## Deployment
Hosted on Render with UptimeRobot.
